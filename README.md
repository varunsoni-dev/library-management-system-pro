# 📚 Library Management System

A console-based Library Management System built with Python using Object-Oriented Programming principles.

This project demonstrates clean project organization, modular design, JSON-based data persistence, exception handling, and core Python programming concepts.

---

## ✨ Features

- Add new books
- Remove books
- Search books by ID
- Search books by Title
- Borrow books
- Return books
- Display all books
- Count total books
- Save data automatically using JSON
- Load saved books on startup
- Exception handling for invalid input

---

## 🛠 Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- JSON
- File Handling
- Exception Handling
